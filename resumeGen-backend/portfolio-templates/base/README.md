# Your Portfolio Website

A personal portfolio website generated by **ResumeGen**.

## Quick Start

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Run locally**

   ```bash
   npm run dev
   ```

   Open [http://localhost:5173](http://localhost:5173) in your browser.

3. **Build for production**
   ```bash
   npm run build
   ```
   The output will be in the `dist/` folder — ready to deploy.

## Deploying

### Vercel (recommended)

```bash
npm i -g vercel
npm run build
vercel --prod
```

### Netlify

```bash
npm i -g netlify-cli
npm run build
netlify deploy --prod --dir=dist
```

### Any static host

Run `npm run build` and upload the `dist/` folder to any static hosting provider (GitHub Pages, Cloudflare Pages, etc.).

## Customizing

- **Data** — Edit `src/data.js` to update your resume information
- **Theme** — The theme is set in `src/App.jsx` (change the import and `THEME` constant)
- **Styles** — Theme CSS files are in `src/themes/`
